- human-readable
- 1033 bytes in size
- not executable

## Notas adicionales
El comando find tiene algunos parámetros para ayudar a encontrar lo que buscamos:
+ -readable sirve para encontrar archivos que son leíbles.
+ -type sirve para definir el tipo de archivo que queramos encontrar.
+ -size sirve para definir el tamaño del archivo que queremos encontrar.
